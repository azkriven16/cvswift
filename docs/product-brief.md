# Product Brief

## Name

CVSwift

## Positioning

An open-source resume builder with real hosted accounts, private resume storage, and free AI-powered resume audits.

## Product Promise

Build resumes that survive the first screen.

## Audience

- Frontend engineers
- Product engineers
- Design engineers
- Developers who want an open-source resume tool
- Job seekers who want lightweight resume feedback without a paywall

## Hosted Free Product

Free users get:

- 5 cloud resumes
- 2 AI audits per day
- Basic resume templates
- PDF export generated on demand
- Optional source resume uploads with strict size limits

## Future Paid Product

Pro is not required for the initial free portfolio version.

If added later, Pro can be $5/month and include:

- More than 5 cloud resumes
- Higher AI audit limits
- AI bullet rewrites
- Job-description tailoring
- Premium templates
- Version history

## Open-Source Product

The repo should stay useful for self-hosters:

- Users bring their own Supabase project
- Users bring their own OpenRouter or compatible AI key
- No hard dependency on paid services
- Clear setup docs and SQL schema

## Non-Goals For MVP

- No stored generated PDFs
- No paid Stripe flow yet
- No permanent file storage beyond optional uploaded source documents
- No complex team/workspace model
- No admin dashboard
